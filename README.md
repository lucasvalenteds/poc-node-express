# POC: Express

It demonstrates how to structure a [REST API](https://en.wikipedia.org/wiki/Representational_state_transfer) implemented using [Express](https://github.com/expressjs) framework targeting Node.js.

## How to run

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
| Run server | `npm start` |
